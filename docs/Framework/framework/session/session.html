Class Session

Session

public class Session

Session Handling Class

Sessions can be a bit of a pain to handle as arrays, this class gives access to the _SESSION super global through a class interface. This class is entirely static due to the global nature of the sessions in PHP.

Once Variables

This class has the notion of once variables. That is variables that are stored in the session until the end of the next session (page load). To do this the class uses a singleton-like pattern to take advantage of the __destruct functionality.
